### Installing

To start the project install all the dependencies.

```
npm install
```

or

```
yarn install
```

## Getting Started
To install the web pack, all you need to do is using npm or yarn to start it.
```
npm start
```
or 

```
yarn start
```

* You can checkout the web app here: [http://localhost:8080](http://localhost:8080)
